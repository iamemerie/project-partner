# MERN Application

**Lab Phase Project — GoMyCode**

**An educational ecosystem application built using the MERN stack (MongoDB, Express.js, React.js, Node.js) designed to streamline academic management, resource sharing, and role-based user experiences.**

# Project Topic

**Academicaids.ng — Tutoring and Scholarship Resource Platform**

Stack
Frontend: React.js (with dynamic state management and responsive UI layout)

Backend: Node.js & Express.js (RESTful API architecture)

Database: MongoDB Atlas (Mongoose ODM)

Authentication: Google OAuth 2.0 Integration

Core Features Implemented
Dynamic Role-Based Access Control (RBAC): Secure user management handling distinct Student, Tutor, and Admin dashboards, dynamically driven by live database state changes.

Secure Third-Party Authentication: Streamlined user onboarding via Google OAuth integration on both the frontend and backend.

Cloud Data Persistence: Fully integrated with MongoDB Atlas for scalable, cloud-hosted data management.
